# Ipea Demo

**I have a Ipea** · 点亮身边的好活动

广州本地活动发现 App 的课堂演示原型。基于地图浏览周边活动，支持筛选、推荐、想去、打卡与评价等核心流程（演示用 mock 数据）。

## 在线预览

部署后访问：**https://sq-chen.github.io/ipea-demo/**

> V1 已上线：广州示意地图 + 12 个 mock 活动，点击标记弹出卡片。

## 本地预览

直接用浏览器打开 `index.html`，或使用本地静态服务器：

```bash
# Python 3
python -m http.server 8080
# 浏览器访问 http://localhost:8080
```

## 技术说明

- 静态 HTML + CSS + JavaScript
- GitHub Pages 托管，手机 / 电脑浏览器均可访问
- 静态广州地图 + 本地 mock 数据，完全离线可用
